## Screen-by-Screen Explanation
# Splash Screen
Displays the SkillUp logo and tagline “Rise with Skills”, creating a strong first impression and brand identity.
# Login Screens
Users can log in using email or mobile number.
Forgot password and social login improve accessibility.
# Signup Screen
Simple form layout prevents user input errors.
# Forgot Password & Email Sent
These screens reassure users and support account recovery.
# Home Screen
Includes:
•	welcome message
•	search bar
•	categories
•	popular courses
•	continue learning section
This functions as the learning dashboard.
# Course Listing
Courses are shown in card format with titles, instructor names, and duration.
# Course Details
Includes:
•	course thumbnail
•	instructor name
•	description
•	duration
•	enroll /continue button
# Course Outline
Shows module list and progress bar, helping learners track learning.
# Video Player
Minimal design to keep focus on learning content.
# Profile Section
Contains:
•	My Courses
•	Certificates
•	Wishlist
•	Notifications
•	Help & Support
•	Logout
# Help & Support
Provides FAQ, contact options, feedback, and ratings.
# Certificates Page
Displays earned certificates to motivate progress.
# Logout Confirmation
Prevents accidental sign-out.
